Adds a print link to WooCommerce reports

== Description ==

Without this plugin WooCommerce report graphs are not easily printable. This plugin scales the report graph down to a printable size and formats the admin page to make it more printer friendly.

== Installation ==

[WooCommerce](https://wordpress.org/plugins/woocommerce/ "WooCommerce plugin page") is required to work with this plugin.

1. Install this (WooCommerce Print Report) plugin
1. When logged into WordPress while this plugin and WooCommerce are both active, navigate to WooCommerce -> Reports
1. While viewing a report, a new "Print Report" button should appear next to export on the right-hand side above the graph.
